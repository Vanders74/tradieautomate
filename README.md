# TradieAutomate

Helping Australian solar, battery & electrical installers automate compliance, streamline operations, and grow smarter.

**Built with [Astro](https://astro.build)** — ships zero JavaScript, sub-second page loads.

## Local Development

```bash
npm install
npm run dev
```

## Build & Deploy

Deployed automatically via Netlify on push to `main`.

```bash
npm run build    # builds to ./dist
npm run preview  # preview the build
```

## Structure

```
src/
├── content/blog/    # Blog posts (Markdown/MDX)
├── components/      # Astro components
├── layouts/         # Page layouts
├── pages/           # Routes
└── styles/          # Global CSS
```
